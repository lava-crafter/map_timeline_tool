Privacy Policy for Map Timeline Tool
Map Timeline Tool requires access to the microphone (android.permission.RECORD_AUDIO) solely for the purpose of measuring environmental noise levels (in decibels) when you save a location point.
Data Handling: The audio stream is processed locally in real-time on your device to calculate a numeric noise level value. We do not record human speech, save audio files, or transmit any audio data to remote servers. All audio data is immediately discarded from memory once the calculation is complete.
By using the application, you agree to this data usage policy.
